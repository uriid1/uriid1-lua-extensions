tobits (num[, opts])
Convert a number to its binary representation Precision limited to 2^50

Parameters:

  • num number Input value
  • opts table Options
    • bits number Force output bit width (optional)
    • to_array boolean Return bit array instead of string (optional)

Returns:

    string|table Binary representation as string or array

Usage:

    local bits = M.tobits(255) -- "11111111"
tonum (bin[, opts])
Convert a binary sequence (string or array) to a number Precision limited to 2^50

Parameters:

  • bin string|table Binary data
  • opts table Options
    • bits number Bit length (optional)
    • from_array boolean Interpret input as bit array (optional)

Returns:

    number Decimal value

Usage:

    local num = M.tonum("11111111") -- 255
getBytes (num)
Extract bytes from a number Precision: up to 6 bytes (48 bits)

Parameters:

  • num number Input value

Returns:

    table Array of bytes (most significant first)

Usage:

    local bytes = M.getBytes(0xAAFFCCDDA1C1)
for i = 1, #bytes do
  print(string.format("Byte %d: 0x%02X", i, bytes[i]))
